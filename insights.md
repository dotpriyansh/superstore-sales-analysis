# Business Insights

## Customer Revenue Distribution

The analysis shows a significant gap between high-value and low-value customers.

The highest-performing customer, Sean Miller, generated $25,043.05 in sales, while the lowest-performing customer, Thais Sissman, generated only $4.83.

This indicates a highly uneven sales distribution where a relatively small number of customers contribute a substantial portion of overall revenue.

## One-Time Customers

The dataset contains 12 customers who placed only one order.

One-time customers may represent:

* Lost customers
* New customers with limited purchase history
* Customers requiring targeted retention campaigns

Improving repeat purchases from these customers could positively impact long-term revenue.

## SQL and Data Engineering Learnings

This project demonstrates:

* Data normalization using separate customer, product, and order tables
* Data transformation using SQL
* Business analysis using CTEs and Subqueries
* Customer segmentation using Window Functions
* Ranking and performance analysis using analytical SQL

The project highlights how raw transactional data can be transformed into meaningful business insights through SQL-based analysis.
